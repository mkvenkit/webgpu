The Book of WebGPU is a fun programming book that teaches you the fundamentals of 3D graphics programming and GPU computing using WebGPU and JavaScript. This webpage is a live demonstration of the WebGPU examples in the book. I will be updating this page as I make progress with the book. Please use the button on the right to pre-order the book from No Starch Press.

Browser Compatibility

The examples in the book work with Google Chrome 113 and above. WebGPU is gathering browser support rapidly, so please check the official implementation status for the latest status.
